# Object

Generate object values.

## dictionary

Generate dictionaries containing keys generated using `keyArb` and values generated by `valueArb`.

**Signatures:**

- `fc.dictionary(keyArb, valueArb)`
- `fc.dictionary(keyArb, valueArb, {minKeys?, maxKeys?, size?, noNullPrototype?, depthIdentifier?})`

**with:**

- `keyArb` â€” _arbitrary instance responsible to generate keys_
- `valueArb` â€” _arbitrary instance responsible to generate values_
- `minKeys?` â€” default: `0` â€” _minimal number of keys in the generated instances (included)_
- `maxKeys?` â€” default: `0x7fffffff` [more](/docs/configuration/larger-entries-by-default/#size-explained) â€” _maximal number of keys in the generated instances (included)_
- `size?` â€” default: `undefined` [more](/docs/configuration/larger-entries-by-default/#size-explained) â€” _how large should the generated values be?_
- `noNullPrototype?` â€” default: `true` â€” _only generate objects based on the Object-prototype, do not generate any object with null-prototype_
- `depthIdentifier?` â€” default: `undefined` â€” _share the depth between instances using the same `depthIdentifier`_

## record

Generate records using the incoming arbitraries to generate its values.

It comes very useful when dealing with settings.

**Signatures:**

- `fc.record(recordModel)`
- `fc.record(recordModel, {requiredKeys?, noNullPrototype?})`

**with:**

- `recordModel` â€” _structure of the resulting instance_
- `requiredKeys?` â€” default: `[all keys of recordModel]` â€” _list of keys that should never be deleted_
- `noNullPrototype?` â€” default: `false` â€” _only generate records based on the Object-prototype, do not generate any record with null-prototype_

**Usages:**

```js
fc.record({
  id: fc.uuidV(4),
  age: fc.nat(99),
});
// Examples of generated values:
// â€¢ Object.assign(Object.create(null),{"id":"0000001b-000b-4000-8000-0000ab45c359","age":1})
// â€¢ {"id":"9de2ca04-dc68-4c3f-87e4-e878407827d6","age":98}
// â€¢ {"id":"12bb661f-b57a-4d8c-8407-719100000002","age":99}
// â€¢ {"id":"ffffffe3-0013-4000-a7d5-d5734251f8d2","age":0}
// â€¢ Object.assign(Object.create(null),{"id":"00000007-000a-4000-8000-00170000000e","age":5})
// â€¢ â€¦

fc.record(
  {
    id: fc.uuidV(4),
    age: fc.nat(99),
  },
  { requiredKeys: [] },
);
// Note: Both id and age will be optional values
// Examples of generated values:
// â€¢ Object.assign(Object.create(null),{"id":"00000004-27f6-48bb-8000-000a69064200","age":3})
// â€¢ {"id":"ffffffee-ffef-4fff-8000-0015f69788ee","age":21}
// â€¢ Object.assign(Object.create(null),{"age":34})
// â€¢ Object.assign(Object.create(null),{"id":"2db92e09-3fdc-49e6-8000-001b00000007","age":5})
// â€¢ Object.assign(Object.create(null),{"id":"00000006-0007-4000-8397-86ea00000004"})
// â€¢ â€¦

fc.record(
  {
    id: fc.uuidV(4),
    name: fc.constantFrom('Paul', 'Luis', 'Jane', 'Karen'),
    age: fc.nat(99),
    birthday: fc.date({ min: new Date('1970-01-01T00:00:00.000Z'), max: new Date('2100-12-31T23:59:59.999Z') }),
  },
  { requiredKeys: ['id'] },
);
// Note: All keys except 'id' will be optional values. id has been marked as required.
// Examples of generated values:
// â€¢ {"id":"46045be9-0009-4000-8000-0008ffffffed","name":"Karen","age":11,"birthday":new Date("2100-12-31T23:59:59.997Z")}
// â€¢ Object.assign(Object.create(null),{"id":"fffffffe-0015-4000-95a0-f8e9ffffffe7","name":"Karen","birthday":new Date("1970-01-01T00:00:00.018Z")})
// â€¢ {"id":"e2b066ec-000b-4000-bfff-ffe7ccb1828d","name":"Karen","age":17}
// â€¢ {"id":"43b7d8e5-d043-42ef-8000-001a00000005","age":16,"birthday":new Date("2004-10-16T22:01:09.416Z")}
// â€¢ Object.assign(Object.create(null),{"id":"00000007-2008-452e-8000-00133ed36be7","name":"Karen","age":6,"birthday":new Date("2100-12-31T23:59:59.982Z")})
// â€¢ â€¦

fc.record(
  {
    id: fc.uuidV(4),
    age: fc.nat(99),
  },
  { requiredKeys: [] },
);
// Note: Both id and age will be optional values
// Examples of generated values:
// â€¢ Object.assign(Object.create(null),{"id":"00000004-27f6-48bb-8000-000a69064200","age":3})
// â€¢ {"id":"ffffffee-ffef-4fff-8000-0015f69788ee","age":21}
// â€¢ Object.assign(Object.create(null),{"age":34})
// â€¢ Object.assign(Object.create(null),{"id":"2db92e09-3fdc-49e6-8000-001b00000007","age":5})
// â€¢ Object.assign(Object.create(null),{"id":"00000006-0007-4000-8397-86ea00000004"})
// â€¢ â€¦

fc.record(
  {
    id: fc.uuidV(4),
    age: fc.nat(99),
  },
  { noNullPrototype: true },

// Note: If you only want instances coming with the prototype of Object, you can toggle the flag noNullPrototype.
// The prototype of Object carry some extra functions with it: `generatedInstance.toString()` can be achieved on it, it "cannot" without a prototype if no toString was explicitely defined.
// Examples of generated values:
// â€¢ {"id":"b635cdb8-001b-4000-bd7a-8cbfffffffeb","age":79}
// â€¢ {"id":"ffffffec-5522-4356-bc7e-9b14ffffffe5","age":69}

// â€¢ â€¦
```

## object

Generate objects (key/values).

**Signatures:**

- `fc.object()`
- `fc.object({key?, depthSize?, maxDepth?, maxKeys?, size?, withBigInt?, withBoxedValues?, withDate?, withMap?, withNullPrototype?, withObjectString?, withSet?, withTypedArray?, withSparseArray?, withUnicodeString?, stringUnit?, values?})`

**with:**

- `key?` â€” default: `fc.string()` â€” _arbitrary responsible to generate keys used for instances of objects_
- `depthSize?` â€” default: `undefined` [more](/docs/configuration/larger-entries-by-default/#depth-size-explained) â€” _how much we allow our recursive structures to be deep?_
- `maxDepth?` â€” default: `Number.POSITIVE_INFINITY` â€” _maximal depth for generated objects (Map and Set included into objects)_
- `maxKeys?` â€” default: `0x7fffffff` [more](/docs/configuration/larger-entries-by-default/#size-explained) â€” _maximal number of keys in generated objects (Map and Set included into objects)_
- `size?` â€” default: `undefined` [more](/docs/configuration/larger-entries-by-default/#size-explained) â€” _how large should the generated values be?_
- `withBigInt?` â€” default: `false` â€” _enable `bigint` - eg.: `1n`_
- `withBoxedValues?` â€” default: `false` â€” _enable boxed values - eg.: `new Number(5)`_
- `withDate?` â€” default: `false` â€” _enable `Date` - eg.: `new Date('2020-10-14T16:52:36.736Z')`_
- `withMap?` â€” default: `false` â€” _enable `Map` - eg.: `new Map([['key', 'value']])`_
- `withNullPrototype?` â€” default: `false` â€” _enable objects not defining any prototype - eg.: `Object.create(null)`_
- `withObjectString?` â€” default: `false` â€” _enable strings looking as string representations of JavaScript instances - eg.: `"{}"`, `"new Set([1])"`_
- `withSet?` â€” default: `false` â€” _enable `Set` - eg.: `new Set([1, 2, 3])`_
- `withTypedArray?` â€” default: `false` â€” _enable typed arrays for ints, uints and floats - eg.: `Int8Array.from([1, 2, 3])`_
- `withSparseArray?` â€” default: `false` â€” _enable sparse arrays - eg.: `[1,,,3]`_
- `withUnicodeString?` â€” default: `false` â€” _change the default for `key` and `values` so that they produce unicode strings with non-ascii characters â€” shadowed by `stringUnit`_
- `stringUnit?` â€” default: `undefined` â€” _customize the set of characters being used by the `string` arbitrary_
- `values?` â€” default: _booleans, numbers, strings, null and undefined_ â€” _array of arbitraries producing the root* values - *non-object ones_

## anything

Generate any kind of entities.

**Signatures:**

- `fc.anything()`
- `fc.anything({key?, depthSize?, maxDepth?, maxKeys?, size?, withBigInt?, withBoxedValues?, withDate?, withMap?, withNullPrototype?, withObjectString?, withSet?, withTypedArray?, withSparseArray?, withUnicodeString?, stringUnit?, values?})`
